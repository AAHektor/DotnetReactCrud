# DotnetReactCrud

A fullstack application for person management built with .NET 10 and React 19.
(Built for fun and for learning purposes)

## Tech Stack & Libraries

### Backend (.NET 10)
- **Framework:** ASP.NET Core 10.0 Web API.
- **Database:** SQLite with Entity Framework Core 10.0.2.
- **Architecture:** Controller-based API with dedicated Models for data structuring.
- **Tooling:** EF Core Design tools for managing database migrations and REST Client (VS Code) for endpoint testing.

### Frontend (React 19 + Vite 7)
- **Framework:** React 19 with Vite 7.
- **Routing:** React Router DOM v7 for client-side navigation.
- **Styling:** Tailwind CSS v4 (using the @tailwindcss/vite plugin).
- **Form Handling:** React Hook Form for efficient data entry and validation.
- **API Client:** Axios for managing HTTP communication with the backend.
- **UI/UX:** Lucide React icons and React Hot Toast for real-time feedback.
